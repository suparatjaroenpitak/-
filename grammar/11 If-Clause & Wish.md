# 10. If-Clause & Wish

### โจทย์มักออกแบบไหน

โจทย์จะให้เลือกกริยาในประโยคเงื่อนไข หรือประโยคแสดงความปรารถนาที่ตรงข้ามกับความจริง

ตัวอย่างรูปแบบโจทย์:

> If I ___ still young, I would climb that mountain.

หรือ

> I wish I ___ too much last night.

### Choice มักมาอย่างไร

ตัวเลือกมักเป็น tense หลายแบบ:

A. am  
B. was  
C. were  
D. will be

หรือ

A. don't eat  
B. didn't eat  
C. hadn't eaten  
D. haven't eaten

### เทคนิคตอบ

If-Clause ที่เจอบ่อยในข้อสอบ ก.พ. คือ Type 2

โครงสร้าง:

> If + S + V.2, S + would + V.1

ใช้กับเรื่องสมมติที่ไม่จริงในปัจจุบัน

ตัวอย่าง:

> If I were rich, I would buy a house.

สรุป If-Clause:

| Type | โครงสร้าง | ใช้เมื่อ |
|---|---|---|
| Type 0 | If + Present, Present | ความจริงทั่วไป |
| Type 1 | If + Present, will + V.1 | เป็นไปได้ในอนาคต |
| Type 2 | If + Past, would + V.1 | สมมติ ไม่จริงในปัจจุบัน |
| Type 3 | If + Past Perfect, would have + V.3 | เสียดายอดีต |

Wish ที่ออกบ่อย:

| ความหมาย | โครงสร้าง |
|---|---|
| อยากให้ปัจจุบันต่างจากความจริง | wish + Past Simple |
| เสียดายอดีต | wish + Past Perfect |

ตัวอย่าง:

- `I wish I were taller.`
- `I wish I hadn't eaten too much.`

ข้อควรระวัง:

- `If I were...` ใช้ได้แม้ประธานเป็น I
- ถ้าเห็น `would + V.1` ในอีกฝั่ง ฝั่ง if มักเป็น V.2
- ถ้าเป็นการเสียดายอดีต ใช้ `had + V.3`

## รูปแบบโจทย์

1. ช่องว่างมักทดสอบโครงสร้าง If-Clause (Type 0–3) หรือ `wish` ที่ใช้รูป Past / Past Perfect
2. ตัวเลือกมักเป็นรูป tense หลายแบบเพื่อทดสอบการแยก Type

## ตัวเลือก (Choice) มักเป็น

- am / was / were / will be
- don't eat / didn't eat / hadn't eaten / haven't eaten

## ตัวอย่างโจทย์และคำตอบ

1) If I ___ still young, I would climb that mountain.  
	- Choices: am / was / were / will be  
	- คำตอบ: `were` (Type 2 — hypothetical present)

2) I wish I ___ too much last night.  
	- Choices: don't eat / didn't eat / hadn't eaten / haven't eaten  
	- คำตอบ: `hadn't eaten` (wish about past — Past Perfect)

3) If it ___ rains, the picnic will be canceled.  
	- Choices: were / doesn't / doesn't / didn't  
	- คำตอบ: `doesn't` (Type 1 / zero? — depends on structure; ensure subject-verb agreement)

## สังเกตสัญญาณรอบช่องว่าง (If-Clause / Wish)

- มอง `would`, `could`, `might` เป็นสัญญาณว่าเป็น type 2 หรือ 3 (hypothetical) — ถ้ามี `would` ใน main clause ฝั่ง if มักเป็น Past Simple (type 2)
- `had + V.3` ใน if-clause → type 3 (past unreal) และ main clause ใช้ `would have + V.3`
- `wish`:
  - `wish + Past Simple` → ปรารถนาให้ปัจจุบันต่างจากความจริง (`I wish I were taller`)
  - `wish + Past Perfect` → เสียดายอดีต (`I wish I hadn't eaten`)
- คำใบ้เวลา: `if + present` กับ `will`/`may` ในผลลัพธ์ → type 1 (real possibility)
- เทคนิคจากครูแมรี่: ถ้าเห็น `were` ใช้ได้แม้ประธานเป็น `I` (formal/hypothetical) — อย่าแทง `was` ในกรณีสมมติที่ต้องการความสุภาพ

### ตัวอย่างเชิงปฏิบัติ

1) `If I ___ still young, I would climb that mountain.` → ถ้าตัวเลือกมี `were` ให้เลือก (type 2 hypothetical present)
2) `I wish I ___ too much last night.` → ถ้าต้องการเสียดายอดีต → `hadn't eaten` (wish + past perfect)
3) `If you heat water, it ___.` → Type 0 (general truth) → `boils`

เคล็ดลับ: ดู modal ใน main clause (`would`, `could`) และ time markers (past perfect) เพื่อแยก Type ได้เร็ว
