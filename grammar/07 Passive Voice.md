# 6. Passive Voice

### โจทย์มักออกแบบไหน

โจทย์จะให้ประธานเป็นสิ่งของหรือคนที่ไม่ได้ทำกริยาเอง แต่ถูกกระทำ แล้วให้เลือกโครงสร้าง passive

ตัวอย่างรูปแบบโจทย์:

> Baseball ___ to Japan many years ago.

หรือ

> Spanish ___ as an official language in many countries.

### Choice มักมาอย่างไร

ตัวเลือกมักผสม active/passive:

A. introduced  
B. was introduced  
C. introducing  
D. has introduced

หรือ

A. uses  
B. used  
C. is used  
D. using

### เทคนิคตอบ

ถามตัวเองว่า "ประธานทำเองได้ไหม"

- ถ้าประธานทำเองได้ อาจเป็น Active
- ถ้าประธานถูกกระทำ ใช้ Passive

โครงสร้างหลัก:

> Verb to be + V.3

ตัวอย่าง:

- `is used`
- `was introduced`
- `are required`
- `has been completed`

เทคนิคเลือก tense:

| เวลา | Passive |
|---|---|
| Present Simple | is/am/are + V.3 |
| Past Simple | was/were + V.3 |
| Present Perfect | has/have been + V.3 |
| Future | will be + V.3 |

ข้อควรระวัง:

- ต้องมีทั้ง verb to be และ V.3
- ถ้ามี `by + ผู้กระทำ` มักเป็นสัญญาณ passive
- สิ่งของ เช่น `baseball`, `Spanish`, `law`, `document` มักถูกกระทำมากกว่าทำเอง

## รูปแบบโจทย์

1. ช่องว่างมักให้เติมรูป passive (be + V.3) หรือรูป active เพื่อทดสอบการแยก active/passive
2. ตัวเลือกมักประกอบด้วยรูป active simple, passive simple, gerund/participle

## ตัวเลือก (Choice) มักเป็น

- introduced / was introduced / introducing / has introduced
- is used / used / uses / using

## ตัวอย่างโจทย์และคำตอบ

1) Baseball ___ to Japan many years ago.  
	- Choices: introduced / was introduced / introducing / has introduced  
	- คำตอบ: `was introduced` (past passive)

2) Spanish ___ as an official language in many countries.  
	- Choices: uses / used / is used / using  
	- คำตอบ: `is used` (present passive)

3) The report ___ by the committee yesterday.  
	- Choices: prepared / was prepared / preparing / has prepared  
	- คำตอบ: `was prepared` (past passive + by phrase)

## สังเกตสัญญาณรอบช่องว่าง (Passive Voice)

- หน้าช่องว่าง: มี `is/are/was/were/has been/have been` หน้า blank → มักเป็น passive (be + V.3)
- หลังช่องว่าง: ถ้ามี `by + ...` หรือถ้าประธานเป็นสิ่งของ → ให้พิจารณา passive
- คำช่วย: ถ้าช่องว่างมีรูป V.3 แต่ไม่มี be ก่อนหน้า อาจเป็น past participle หรือ adjective — เช็กบริบทว่าเป็น passive หรือไม่

### สัญญาณเชิงลึก (จากครูแมรี่)

- มองหา `be + V.3` pattern: ถ้ามี auxiliary เช่น `is/was/has been/will be` ให้คาด passive ทันที
- `by + agent` เป็นสัญญาณชัดเจนของ passive แต่บางครั้ง agent ถูกตัดออก (agent omitted) — ถ้าประธานเป็นสิ่งของ/ภาษา/กฎ มักเป็น passive
- Tense mapping: ตรวจ auxiliary ก่อน (is/are → present passive; was/were → past passive; has/have been → perfect passive)
- Passive reduction: ประโยคที่เป็น reduced relative clause เช่น `the book written by the author` อาจวางเป็น `the book written` ในตัวเลือก — ระวังว่าบางตัวเลือกเป็น participle ที่ทำหน้าที่ adjective

### ตัวอย่างการสังเกตเชิงปฏิบัติ

1) `The report ___ by the committee yesterday.` → มี `yesterday` → past → `was prepared`  
2) `Spanish ___ as an official language in many countries.` → ประธานเป็นภาษา → มักเป็น passive `is used`  
3) `The new policy has ___ implemented.` → ถ้าต่อด้วย V.3 และ `has` ก่อนหน้า → `has been implemented`

เคล็ดลับ: ถ้าตัดช้อยส์แล้วเจอรูป V.3 หลายรูป ให้ย้อนดู auxiliary และ time marker เพื่อเลือก tense ที่ถูกต้อง
