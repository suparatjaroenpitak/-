# Comparison (การเปรียบเทียบ)

## ประเภท
- **Equality (ขั้นเท่า):** as + adj + as → as tall as, as easy as
- **Comparative (ขั้นกว่า):** adj+er / more + adj + than → taller / more expensive than
- **Superlative (ขั้นสุด):** the + adj+est / the most + adj → the tallest / the most important

## กฎพื้นฐาน (นับพยางค์)
- คำ 1 พยางค์: เติม -er / -est (tall → taller / tallest)
- คำ 2 พยางค์: ถ้าลงท้ายด้วย -y ให้เปลี่ยน y → i แล้วเติม -er/-est (happy → happier / happiest)
- คำ 3 พยางค์ขึ้นไป: ใช้ more / most (more beautiful / the most beautiful)

## คีย์เวิร์ดช่วยทำข้อสอบ
- `than` → ขั้นกว่า
- `as ... as` → ขั้นเท่า
- `the` + (`in`/`of`/`has + V3`/`most`) → มักเป็นขั้นสุด

## เทคนิคการทำข้อสอบ
1. มองหา keyword ในโจทย์ (as / than / the / most / in / of)
2. ถ้าเป็นขั้นสุด ให้หา `the` + `-est` หรือ `the most`
3. นับพยางค์คำเปรียบเทียบเพื่อเลือกว่าจะใช้ `-er/-est` หรือ `more/most`

## ตัวอย่าง
- Anna runs as fast as Lisa. (as ... as)
- Nadia's house is the largest one in our neighborhood. (the + -est + in)
- Summer is hotter than winter in Thailand. (comparative)

---

สรุป: หา keyword ก่อน → นับพยางค์ → ใส่รูปแบบที่เหมาะสม
