# 10. If-Clause & Wish

### โจทย์มักออกแบบไหน

โจทย์จะให้เลือกกริยาในประโยคเงื่อนไข หรือประโยคแสดงความปรารถนาที่ตรงข้ามกับความจริง

ตัวอย่างรูปแบบโจทย์:

> If I ___ still young, I would climb that mountain.

หรือ

> I wish I ___ too much last night.

### Choice มักมาอย่างไร

ตัวเลือกมักเป็น tense หลายแบบ:

A. am  
B. was  
C. were  
D. will be

หรือ

A. don't eat  
B. didn't eat  
C. hadn't eaten  
D. haven't eaten

### เทคนิคตอบ

If-Clause ที่เจอบ่อยในข้อสอบ ก.พ. คือ Type 2

โครงสร้าง:

> If + S + V.2, S + would + V.1

ใช้กับเรื่องสมมติที่ไม่จริงในปัจจุบัน

ตัวอย่าง:

> If I were rich, I would buy a house.

สรุป If-Clause:

| Type | โครงสร้าง | ใช้เมื่อ |
|---|---|---|
| Type 0 | If + Present, Present | ความจริงทั่วไป |
| Type 1 | If + Present, will + V.1 | เป็นไปได้ในอนาคต |
| Type 2 | If + Past, would + V.1 | สมมติ ไม่จริงในปัจจุบัน |
| Type 3 | If + Past Perfect, would have + V.3 | เสียดายอดีต |

Wish ที่ออกบ่อย:

| ความหมาย | โครงสร้าง |
|---|---|
| อยากให้ปัจจุบันต่างจากความจริง | wish + Past Simple |
| เสียดายอดีต | wish + Past Perfect |

ตัวอย่าง:

- `I wish I were taller.`
- `I wish I hadn't eaten too much.`

ข้อควรระวัง:

- `If I were...` ใช้ได้แม้ประธานเป็น I
- ถ้าเห็น `would + V.1` ในอีกฝั่ง ฝั่ง if มักเป็น V.2
- ถ้าเป็นการเสียดายอดีต ใช้ `had + V.3`
