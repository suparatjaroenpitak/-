# สรุป Conversation, Reading, Vocabulary - ก.พ. (จากครูแมรี่)

## 📖 READING (10 ข้อ - หนักสุด)

### โจทย์มักออกแบบไหน

1. **Main Idea / Purpose** - ถามถึงใจความสำคัญของบทความ
2. **Vocabulary** - ถามความหมายของคำศัพท์
3. **True / False** - ถามความถูกต้องตามข้อมูล
4. **Question Words (QW)** - ถามรายละเอียดเฉพาะ
   - What = สิ่งของ, การกระทำ
   - Who = คน
   - Where = สถานที่
   - Why = เหตุผล (ตามด้วย because)
   - When = เวลา
   - Which = การเลือก
   - How = วิธีการ
5. **Pronoun Reference** - ถามว่าคำแทน (it, they, etc.) หมายถึงใครหรือสิ่งใด

### Choice มักมาอย่างไร

| ประเภท | ลักษณะ |
|---|---|
| Main Idea | ตัวเลือกเป็นประโยคสรุป/บทสรุป 4-5 ข้อ |
| Vocabulary | ตัวเลือกเป็นคำศัพท์คล้ายความหมาย 4-5 คำ |
| True/False | ข้อมูลตรงกับบทความ หรือผิดบ้าง |
| QW Questions | ตัวเลือกเป็นข้อมูลรายละเอียด 4-5 รายการ |

### เทคนิคตอบ (ลำดับการทำ)

#### ✅ ขั้นที่ 1: อย่าอ่านบทความทั้งหมด
- **เสียเวลา** ❌
- เขาให้ประมาณ 5 ข้อคำถาม
- อ่านคำถามก่อน เพื่อรู้ว่าต้องโฟกัสตรงไหน

#### ✅ ขั้นที่ 2: อ่านคำถามก่อน
```
สมมติถามว่า: "What is the purpose of this meeting?"
→ โฟกัสที่ประโยคแรก (Introduction)
```

#### ✅ ขั้นที่ 3: หาจุดโฟกัส (Key Information)
- **Main Idea** → ประโยคที่ 1-2
- **Vocabulary** → ดูคำใบ้ด้านหลังหรือข้างหน้า
- **Question Words** → ตามตัวโจทย์
- **True/False** → ข้ามได้ถ้าไม่ถนัด (ต้องอ่านทั้งหมด)

#### ✅ ขั้นที่ 4: เทียบคีย์เวิร์ด (Keyword Matching)
```
โจทย์: "The email is used for..."
→ ตามหา "used for" ในบทความ
→ เทียบตัวเลือกกับส่วนนั้น
```

#### ✅ ขั้นที่ 5: ตัดตัวเลือกที่ไม่ตรง
- ไม่ต้องแปลทั้งหมด
- สังเกตคำเหมือนกับโจทย์
- เลือกข้อที่มีคำใบ้เยอะที่สุด

### ตัวอย่าง: How long can leftover food be stored?

| Choice | Analysis | ✓/✗ |
|---|---|---|
| Up to one month | มีคำ "store" | ✓ แต่ไม่ตรงเวลา |
| Up to 2 months | มีคำ "store" | ✓ แต่ไม่ตรง |
| **Up to 4 months** | **มี "4 months" ตรงกับบทความ** | **✓✓** |
| Increase in size | ไม่มี "store" หรือ "time" | ✗ |

### ข้อสังเกตสำคัญ

1. **ไม่ต้องแปลเต็มบทความ** ⏱️
   - อ่านแบบ skimming (กวาดสายตา)
   - หาคำเหมือน keyword matching

2. **Main Idea → ดูประโยคแรก**
   - มักบอกใจความในบรรทัดแรก
   - ส่วนใหญ่จบด้วย period (.)

3. **Vocabulary → ใช้ Context Clues**
   - ดูคำข้างหลังคำนั้น
   - บางครั้งมีคำใบ้เหมือนกัน
   - ไม่จำเป็นต้องรู้ทั้งคำศัพท์ทั้ง 4 ตัวเลือก

4. **Pronoun (it, they, etc.)**
   - ย้อนไปด้านหน้า หาคำนาม
   - it/he/she/they = สิ่งของ/คน
   - ทำไมคำที่ครูมาร์คไว้หรือหาคำนามใกล้เคียง

5. **Question Word ต้องตรงกัน**
   - Where → สถานที่
   - Why → เหตุผล + because
   - How → วิธีการ
   - เลือกข้อที่ตรงทิศทางที่สุด

## 🎤 CONVERSATION (5 ข้อ)

### โจทย์มักออกแบบไหน

1. **Phone Calls** (โทรศัพท์) - บ่อยที่สุด
   - แจ้งข้อมูล
   - ขอความช่วยเหลือ
   - ส่งข้อความไปหาคนอื่น

2. **Job Interviews** (สัมภาษณ์งาน)
   - ถาม-ตอบประวัติ
   - ถาม-ตอบความสามารถ

3. **Asking for Directions** (ถามทาง)
   - ถามสถานที่
   - ถามเส้นทาง

### Choice มักมาอย่างไร

ตัวเลือก 4 ข้อสรุป:
- 1 ข้อ ถูกต้องเล่
- 2 ข้อ อาจจะใช้ได้ในบริบทอื่น
- 1 ข้อ ไม่เกี่ยวกับบริบท

### เทคนิคตอบ

#### 🔍 **Step 1: สังเกต Type of Question**

| Question Type | คำขึ้นต้น | ตอบแบบ |
|---|---|---|
| **Yes/No Question** | be, do, have, modal verb | Yes/No + ข้อมูล |
| **Question Word (QW)** | What, Who, Where, Why, When, How | ข้อมูล (ไม่ใช่ Yes/No) |

**ตัวอย่าง:**
```
❌ "Did you go to hospital?" 
✓ ตอบ: "Yes, I did" (ถ้าใช่)

✓ "Why did you go?"
✓ ตอบ: "Because I was sick" (ห้ามตอบเพียง "Yes")
```

#### 🔍 **Step 2: ตรวจสอบบริบท (Context)**

| สถานการณ์ | คำตอบที่เหมาะ |
|---|---|
| โทรไปโรงเรียน | "How may I help you?" ✓ |
| | "What are you doing?" ✗ |
| สัมภาษณ์งาน | "Good morning, please have a seat" ✓ |
| | "Where are you?" ✗ |
| ขอความช่วยเหลือ | "Thank you for calling. How can I help?" ✓ |
| | "Would you call me later?" ✗ |

#### 🔍 **Step 3: ดูมารยาท (Politeness)**

ถ้ามีหลายตัวเลือกที่ถูกต้อง → เลือกตัวที่เป็นการพูดแบบมีสัมมารยะ

```
❌ "Come here"
✓ "Would you come here, please?" (มีสัมมารยะมากกว่า)
```

### 4 สิ่งสำคัญของ Conversation

1. **Direction of Answer** - ความสัมพันธ์ของคำตอบ
2. **Context** - สถานการณ์
3. **Type of Question** - ประเภทคำถาม
4. **Politeness** - ความสุภาพ

### ตัวอย่าง: Phone Call to School

**Scenario:** โทรไปโรงเรียนเพื่อบอกว่าลูกป่วย

| Choice | Analysis | ✓/✗ |
|---|---|---|
| Where are you? | ไม่เกี่ยวกับบริบท | ✗ |
| What are you doing? | บริบทผิด (บอกลูกป่วย ไม่ใช่ถามกิจกรรม) | ✗ |
| Would you call later? | ตรงข้าม (เขาโทรไปเองแล้ว) | ✗ |
| **How may I help you?** | **มารยาท + ขอข้อมูล** | **✓✓** |

### Pattern ของ Conversation

| Scene | First Line | Response |
|---|---|---|
| Phone received | "Thank you for calling..." | บอกวัตถุประสงค์ |
| Asking info | "What is...?" | ตอบข้อมูล |
| Thank you | "Thank you for..." | "You're welcome" ✓ |
| Interview greeting | "Good morning, please have a seat" | "Good morning, thank you" |

## 📚 VOCABULARY (5 ข้อ)

### โจทย์มักออกแบบไหน

```
"What does the word 'punctual' mean?"
A. Arriving late
B. On time ✓
C. Early
D. Direct
```

### Choice มักมาอย่างไร

- ตัวเลือก 4 คำ ที่มีความหมายคล้ายหรือเกี่ยวข้อง
- โดยทั่วไป 1 คำถูก, 3 คำผิด

### เทคนิคตอบ

#### 🔍 **Technique 1: Context Clues (คำใบ้)**

**วิธี:** ดูคำที่อยู่ข้างหน้าหรือข้างหลังคำนั้น

```
ตัวอย่าง: "We have a tight agenda"
tight = แน่น → punctual = ต้องตรงเวลา
→ ไม่ได้ "early" (มาก่อน) ✗
→ ไม่ได้ "late" (มาสาย) ✗
```

#### 🔍 **Technique 2: Process of Elimination**

```
"punctual" คืออะไร?

A. Arriving late ✗ (ตรงข้าม)
B. Early ✗ (ถ้ามาก่อน → คนอื่นรอ)
C. Direct ✗ (ไม่เกี่ยวกับการมา)
D. On time ✓ (ตรงต่อเวลา = punctual)
```

#### 🔍 **Technique 3: Word Association**

| คำศัพท์ | ความหมาย | ใบ้ในบทความ |
|---|---|---|
| punctual | ตรงต่อเวลา | "on time", "tight schedule" |
| urgent | เร่งด่วน | "immediately", "ASAP" |
| favorable | ดี, น่าพอใจ | "good", "excellent" |
| inquiries | คำถาม, การสอบถาม | "ask", "question" |

### ข้อสังเกตสำคัญ

1. **ไม่ต้องท่องศัพท์หมด** 📌
   - ใช้ Context Clues (คำใบ้)
   - ไม่รู้ตัวเลือก ก็หาจากประโยครอบข้าง

2. **Read Around the Word** 👀
   - ดูคำศัพท์ข้างหน้า
   - ดูคำศัพท์ข้างหลัง
   - คำตอบส่วนใหญ่ซ่อนอยู่ตรงนั้น

3. **ใช้ Logic** 💡
   ```
   "She was punctual to the meeting"
   → punctual = บวก (ดี) ✓
   → ไม่ใช่ "late" (ลบ) ✗
   ```

## 📊 สรุปเทคนิครวม

### **READING**
- ✓ อ่านคำถามก่อน
- ✓ Keyword matching (หาคำเหมือน)
- ✓ ไม่ต้องแปลทั้งหมด
- ✓ เทียบตัวเลือก
- ✓ เก็บ 3-4 ข้อ (skip True/False)

### **CONVERSATION**
- ✓ ดู verb ขึ้นต้น (Yes/No หรือ QW)
- ✓ ตรวจสอบบริบท
- ✓ เลือกที่มีมารยาท
- ✓ ทิศทางคำตอบต้องตรง

### **VOCABULARY**
- ✓ ใช้ Context Clues
- ✓ อ่านคำข้างหลัง/ข้างหน้า
- ✓ Process of Elimination
- ✓ Word Association
